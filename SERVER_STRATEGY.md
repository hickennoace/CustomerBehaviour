## חלק א: על Craftiverse — מה זה ואיך זה עובד

### מה זה Craftiverse?

Craftiverse הוא שרת Minecraft תחרותי המציע שני מצבי משחק עיקריים בו-זמנית: **Factions** ו-**Skyblock**. השרת פועל כרשת (Network) — שחקנים מחזיקים פרופיל אחיד עם דרגה, נקודות וסטטיסטיקות, ויכולים לעבור בין המצבים בחופשיות.

הכנסות השרת מגיעות מ**חנות הרשת (Webstore)** — פלטפורמה שבה שחקנים רוכשים יתרונות, קוסמטיקה ומשאבים במשחק בכסף אמיתי. זהו מודל F2P (Free-to-Play) קלאסי.

---

### מצב Factions — כיצד הוא עובד?

**הקונספט:** Factions הוא מצב PvP (שחקן-נגד-שחקן) שבו קבוצות מתחרות על שליטה, משאבים וכבוד. כל Faction תובע קרקע, בונה בסיסים מוגנים ופושט על אויבים.

| מנגנון | הסבר |
|:---|:---|
| **Claiming** | קבוצה תובעת Chunks (16×16 בלוקים) שרק חבריה יכולים לבנות בהם |
| **Power System** | כל שחקן מחזיק "כוח" לפי הרגים ומוות — ללא כוח מספיק אי-אפשר לתבוע קרקע |
| **Raiding** | כשלFaction נגמר הכוח, בסיסם הופך לניתן לפשיטה — גניבת משאבים |
| **Economy** | שחקנים מוכרים משאבים בשוק הפנימי וצוברים כסף במשחק |
| **Gkits** | ערכות ציוד מוכנות שניתן לרכוש — מאפשרות התחלה חזקה יותר |

**מה שחקני Factions קונים:** Gkits (Kit Oracle, Kit Mythic), Fix all, Ranks, Maximum land size.

---

### מצב Skyblock — כיצד הוא עובד?

**הקונספט:** כל שחקן מתחיל על **איי צף קטן** עם משאבים מינימליים. המטרה: לבנות את האי, לגדל משאבים ולמכור בשוק הכלכלי הפנימי של השרת כדי לעלות בדרגה.

| מנגנון | הסבר |
|:---|:---|
| **Island** | כל שחקן מחזיק באי פרטי שניתן לבנות ולשדרג |
| **Farming** | גידול ומכירת משאבים (חיטה, קני סוכר, קקטוס) לרווח כלכלי |
| **Sellwands** | פריט מיוחד שמוכר תכולת חזה שלם בלחיצה אחת עם בונוס כפל הכנסה |
| **Maximum Land Size** | הגדלת שטח האי לאפשר יותר Farm ויותר הכנסה |
| **Island Level** | רמת האי נקבעת לפי הבלוקים שהנחת — גבוהה יותר = יותר יתרונות |

**מה שחקני Skyblock קונים:** Sellwands (Tier1-Tier4), Maximum land size, Fly command, Crate Keys.

## חלק ב: "לפני" — מצב נוכחי

### כיצד אנו מעודדים רכישה כיום?

כיום Craftiverse משתמש בשיטות עידוד בסיסיות ופסיביות — ממתינות שהשחקן יגיע לחנות מעצמו.

**1. נראות יתרונות ה-Rank במשחק**
שחקנים עם דרגת Legend ו-MVP מצוידים ביכולות גלויות (טיסה, כינויים צבעוניים, Prefix בצ'אט). שחקנים אחרים רואים ורוצים — Aspirational Marketing פסיבי.

**2. סיסטם הצבעות (Vote Rewards)**
שחקנים מצביעים לשרת באתרי רישום ומקבלים פרסים (Common Crate Keys). מגדיל נראות השרת ויוצר מעגל חיובי: הצבעות → שחקנים חדשים → הכנסה.

**3. קוסמטיקה בסיסית**
Tags ו-Prefixes בצ'אט מאפשרים לשחקנים לבלוט. "Random tag" ב-$5 הוא מוצר Entry Level.

**4. אירועי מכירה תקופתיים**
"20% OFF all Ranks this weekend!" יוצר Peak של רכישות, אך גם מלמד שחקנים לחכות למכירה הבאה.

**5. פרסומות in-game**
סטאף מדביר פרסומות בצ'אט: *"קנה Legend וקבל Fly! בקר ב-store.craftiverse.net"* — פשוט, לא ממוקד.

### אסטרטגיה 2: טירגוט שחקני Default — Sleeper VIP Activation

**הנתון:**
- 482 שחקני Default מייצרים $6,072.50 — **הכי הרבה מכל דרגה**
- ממוצע הוצאה: $12.60 לשחקן
- 173 שחקנים לא-משלמים — חלקם Default עם מעורבות גבוהה

**מי הם "Sleeper VIPs"?**
Default players הנמצאים בטופ-30% הצבעות + שעות משחק, אך עם הוצאה $0. הם **אוהבים את השרת**, הם **מעורבים**, אך מעולם לא עברו את מחסום הרכישה הראשונה.

**זיהוי Sleeper VIPs (SQL):**
```sql
SELECT username, total_votes, total_playtime_hours, webstore_visits
FROM players_data
WHERE rank = 'Default' AND total_spent_dollars = 0
  AND total_votes >= (SELECT PERCENTILE_CONT(0.7) WITHIN GROUP 
                      (ORDER BY total_votes) FROM players_data)
  AND total_playtime_hours >= (SELECT PERCENTILE_CONT(0.7) WITHIN GROUP 
                               (ORDER BY total_playtime_hours) FROM players_data)
ORDER BY total_votes DESC;
```

**הצעה ממוקדת — "Loyal Player Bundle":**

```
DM Discord אישי לכל Sleeper VIP:

"היי [שם]! שמנו לב שאתה אחד השחקנים הנאמנים ביותר ב-Craftiverse
(הצבעת [X] פעמים — תודה רבה!). בתור הוקרה אישית מאיתנו:

🎁 'Loyal Player Bundle' — בלעדי לך:
  • VIP Rank (שווי $10)
  • Legendary Crate Key (שווי $10)

💰 הכל ב-$12 בלבד — חיסכון של 40%

⏰ ההצעה תקפה 72 שעות בלבד."
```

**מדוע זה עובד:**
- Personalization ("הצבעת [X] פעמים") מראה שאתה **רואה** אותם כאנשים
- "Loyal Player Bundle" מייצר תחושת הכרת תודה, לא מכירה
- 72 שעות עם מועד סיום מוגדר = urgency אמיתי ולא מלאכותי

### אסטרטגיה 3: ניצול פסיכולוגיית FOMO

**הנתון:** המוצרים הננטשים ביותר הם בדיוק המוצרים ב-$10-$25. שחקנים **רוצים** (הוסיפו לעגלה) אך משהו עוצר אותם ברגע האחרון — ספק, ירידת עניין, או "אולי מחר".

**FOMO = Fear of Missing Out** — פחד מהחמצה. שחקני Minecraft בגיל 13-25 רגישים ביותר למנגנון זה.

**יישומים קונקרטיים בחנות:**

**1. Live Purchase Ticker:**
```
"✅ xXDragonSlayer purchased Legend — just now"
"✅ FireMaster purchased Tier4 Sellwand — 2 minutes ago"
"⚡ 847 players online right now"
```

**2. "X שחקנים קנו השבוע" — על כל מוצר:**
```
[Tier4 Sellwand — $12]
🔥 Purchased by 38 players this week
```

**3. הגבלת מלאי וירטואלית:**
```
[MVP Rank — $25]
⚠️ Only 5 upgrade slots available at this price
[Countdown: 48:00:00]
```

**4. Aspiration Banner לדרגת Legend:**
```
🏆 Join the top 5.9% of Craftiverse players.
   12 players upgraded to Legend this week.
   [UPGRADE NOW]
```

**5. Seasonal Limited Items:**
Summer Crate Key, Halloween Box, Winter Box — חוזרים רק פעם בשנה. הבלעדיות עצמה מניעה רכישה. מחיר $7.50 ממלא את הפער בין Common ($5) ל-Legendary ($10).

### אסטרטגיה 4: Bundle Strategy — מכירות משולבות

**הנתון:** Perks מייצרות $3,355 (הכי הרבה), אך שחקן קונה לרוב Perk אחד בנפרד. Bundle מגדיל את ה-AOV (Average Order Value) פי 2-3 מעסקה אחת.

**Bundle מוצעים:**

**1. "Skyblock Starter Pack" — $25 (שווי: $37)**
```
✓ Tier2 Sellwand      ($7)
✓ Fly command         ($10)
✓ Common Crate Key ×2 ($10)
✓ Kit pro             ($10)
─────────────────────────────
  שווי: $37 | מחיר: $25 | חיסכון: 32%
```

**2. "Factions Warrior Pack" — $30 (שווי: $40)**
```
✓ Kit oracle          ($10)
✓ Fix all             ($15)
✓ Maximum land size   ($15)
─────────────────────────────
  שווי: $40 | מחיר: $30 | חיסכון: 25%
```

**3. "Perk Power Bundle" — $35 (שווי: $50)**
```
✓ Fix all             ($15)
✓ Fly command         ($10)
✓ Maximum land size   ($15)
✓ Unban Token         ($10)
─────────────────────────────
  שווי: $50 | מחיר: $35 | חיסכון: 30%
```

**4. "Rank Up + Bonus" — עם כל שדרוג דרגה:**
```
VIP → MVP    = $15 + Legendary Crate Key (בחינם)
MVP → Legend = $25 + Fix all (בחינם)
```

**מדוע Bundles עובדים:**
- **Anchoring:** "שווי $50" גורם ל-$35 להיראות הגיוני
- **Simplification:** אין צורך לחשוב — חבילה מוכנה = פחות Friction
- **AOV:** במקום $10, שחקן מוציא $30 — זינוק של ×3 ב-ARPU מעסקה אחת
